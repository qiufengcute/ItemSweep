# Contribution Guidelines

When submitting a Pull Request to the **`fabric` branch**:

1.  **Do NOT submit raw Java source files!**
2.  **Do NOT submit raw Java source files!**
3.  **Do NOT submit raw Java source files!**

**(Important things said three times.)**

This mod is created and maintained using **MCreator**. Any code modifications to the mod version must be submitted as **MCreator project files (or a complete project exported from it)**.
PRs containing `.java` source files not managed by MCreator **will NOT be merged** (though they may be kept open for discussion).

**Strong Recommendation**: Before starting work, please open an [Issue](./issues) to discuss your proposed changes.

*Thank you for your understanding and cooperation in maintaining the project's core engineering consistency.*
